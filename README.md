# Secure Login System

A secure login system built with Node.js, Express.js, SQLite, bcrypt, and Express Session.

## Features

- User Registration
- User Login
- Password Hashing (bcrypt)
- SQLite Database
- Session Management
- Logout
- SQL Injection Protection
- Input Validation

## Technologies

- Node.js
- Express.js
- SQLite
- bcrypt
- EJS
- Express Session

## Installation

```bash
npm install
```

Initialize the database:

```bash
node initDB.js
```

Run the application:

```bash
node server.js
```

Open:

```
http://localhost:3000
```
